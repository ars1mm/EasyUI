- [EUI_DrawEllipse](ellipse.md) - Draw an ellipse

- [EUI_DrawPolygon](polygon.md) - Draw a polygon

- [EUI_DrawArc](arc.md) - Draw an arc
- [EUI_DrawArcEx](arc_ex.md) - Draw a styled arc

## Text Drawing
- [EUI_DrawText](text.md) - Draw text
